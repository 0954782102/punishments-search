Реклама / Приклади коду

Тут можна вставляти свої приклади коду, і вони будуть гарно підсвічуватись:

HTML приклад:

<div class="logo">
  <img src="logo.png" alt="Furry Logo">
</div>

JavaScript приклад:

document.getElementById("btn").addEventListener("click", () => {
  alert("Привіт з реклами!");
});
